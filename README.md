# Simple User Management API

A basic RESTful API built with **Node.js** and **Express.js** to manage users.  
Supports creating, reading, and deleting users.

## Features
- Add new users (`POST /users`)
- Get all users (`GET /users`)
- Delete a user by ID (`DELETE /users/:id`)
- Check server status (`GET /`)

## Tech Stack
- Node.js
- Express.js
- JavaScript

## How to Run
1. Clone the repo: `git clone <repo-url>`
2. Install dependencies: `yarn install`
3. Start server: `node index.js`
4. Test APIs using Postman or Thunder Client
